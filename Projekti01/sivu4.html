DOM-HARJOITUKSIA

DOM01. Kirjoita JavaScript-koodi, joka muokkaa id="muokattava" tekstin väriä (red), kokoa (2em) ja fonttia (Courier)

HTML + JavaScript

                    HTML:
                        <html>
                        <head>
                            <script src="testi.js"></script>
                        </head>
                        <body>
                            <p id="muokattava">Muokkaa tätä tekstiä</p>
                            <button onclick="muokkaa()">Muokkaa</button>
                        </body>
                        </html>

                    JavaScript
                        function muokkaa() {
                        var teksti = document.getElementById("muokattava");
                        teksti.style.fontFamily = "Courier";
                        teksti.style.color = "red";
                        teksti.style.fontSize = "2em";
                        }
DOM02. Kirjoita JavaScript-koodi, joka hakee etunimen ja sukunimen lomakkeelta ja tulostaa ne konsoliin

HTML + JavaScript

                    HTML:
                    <html>
                    <head>
                      <script src="testi.js"></script>
                    </head>
                    <body>
                        <label>Etunimi: </label><br />
                        <input type="type" name="name" value="" id="etunimi" /><br />
                        <label>Sukunimi: </label><br />
                        <input type="type" name="name" value="" id="sukunimi" /><br />
                        <button onclick="haeTiedot()">Hae tiedot</button>
                    </body>
                    </html>

                    JavaScript
                        function haeTiedot() {
                        var etunimi = document.getElementById("etunimi").value;
                        var sukunimi = document.getElementById("sukunimi").value;
                        console.log("Etunimi on: " + etunimi + " ja sukunimi on: " + sukunimi);
}
