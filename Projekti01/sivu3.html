VE02. Etsi virhe koodista ja korjaa se

                          console.log(message);
                        }

                        printMessage();

VE04. Etsi virhe koodista ja korjaa se 
                        const num1 = 5;
                        const num2 = "3";

                        const sum = num1 + num2;

                        const sum = num1 + parseInt(num2);
                        console.log(sum);

VE06. Etsi virhe koodista ja korjaa se

                        }

                          console.log(count);

VE11. Etsi virhe koodista ja korjaa se

VE12. Etsi virhe koodista ja korjaa se 
                        function add(a, b) {
                          const result = a + b;
                        }

                        const sum = add(3, 4);
                        console.log(sum);

JavaScript

Funktio ei palauta mitään, joten muuttuja sum ei saa arvoa. Lisää return -komento 
                        function add(a, b) {
                          const result = a + b;
                          return result;
                        }

                        const sum = add(3, 4);
                        console.log(sum);
